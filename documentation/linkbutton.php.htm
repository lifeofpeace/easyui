链接按钮 - LinkButton

使用$.fn.linkbutton.defaults重载默认值。

链接按钮用于创建超链接按钮。它是表示正常的<a>标记。它可以显示图标和文字，或只图标或文字。该按钮的宽度可以动态收缩/展开，以适应其文本标签。

使用方法（Usage Example）

创建链接按钮

属性（Properties）

名称 类型 描述 默认值
id string 分配给该组件的ID。 null
disabled boolean 设置为true将禁用按钮。 false
plain boolean 设置为true将显示简洁效果。 false
text string 显示在按钮上的文本。 ''
iconCls string 用来在左边显示一个16x16大小图标的css类。 null

事件（Methods）

名称 属性 描述
options none 返回属性对象。
disable none 禁用按钮。

示例:

$('#btn').linkbutton('disable');
enable none 启用按钮。

示例:

$('#btn').linkbutton('enable');
