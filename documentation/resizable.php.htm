缩放 - Resizable

使用$.fn.resizable.defaults重载默认值。

使用方法（Usage Example）

Create resizable object from markup.

Create resizable object using javascript.


属性（Properties）

名称 类型 描述 默认
disabled boolean 如果设置为true将禁止缩放。 false
handles string 说明缩放的方向，'n'表示向上（北）,'e'表示向右（东）等。 n, e, s, w, ne, se, sw, nw, all
minWidth number 缩放时所允许的最小宽度。 10
minHeight number 缩放时所允许的最小高度。 10
maxWidth number 缩放时所允许的最大宽度。 10000
maxHeight number 缩放时所允许的最大高度。 10000
edge number 将被缩放的边框的边界。 5

事件（Events）

名称 参数 描述
onStartResize e 当开始缩放时触发。
onResize e 在缩放过程中触发，当返回false时, DOM元素将不再缩放。
onStopResize e 当缩放停止是触发。

方法（Methods）

名称 参数 描述
options none 返回可调整大小的选项。
enable none 启用可调整大小的功能。
disable none 禁用可调整大小的功能。
