语法解析 -  Parser

使用方法（Usage）

	$.parser.parse();		// 解析所有的页面
	$.parser.parse('#cc');	// 解析所有的页面

属性（Properties）

名称 类型 描述 默认值
$.parser.auto boolean 定义是否自动解析easyui组件。 true

事件（Events）

名称 参数 描述
$.parser.onComplete context 当语法解析结束时触发事件。

方法（Methods）

名称 参数 描述
$.parser.parse context 解析easyui组件。
