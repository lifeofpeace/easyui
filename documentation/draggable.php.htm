可拖动 - Draggable

使用$.fn.draggable.defaults重载默认值。

 使用方法（Usage Example）

创建一个拖拽元素标记。

使用javascript创建一个拖拽元素。


属性（Properties）

名称 类型 描述 默认值
proxy string,function 拖动时的替代元素，当被设置为'clone'时，将使用该元素的一个复制元素来作为替代元素。如果指定了一个函数, 它将返回一个jquery对象。

下面的例子展示了如何创建一个简单的代理对象。

$('.dragitem').draggable({
	proxy: function(source){
		var p = $('<div style="border:1px solid #ccc;width:80px"></div>');
		p.html($(source).html()).appendTo('body');
		return p;
	}
});
null
revert boolean 如果设置为true, 当拖动停止时元素将返回它的初始位置。 false
cursor string 拖动时的CSS指针。 move
deltaX number 被拖动元素对应于当前指针的水平位置。 null
deltaY number 被拖动元素对应于当前指针的垂直位置。 null
handle selector 开始拖动的句柄。 null
disabled boolean 当设置为true时停止拖动。 false
edge number 可以在其中拖动的容器的宽度 0
axis string 可用值为'v'或者'h',当设置为空时，元素可以同时在水平和垂直方法拖动。 null

事件（Events）

名称 参数 描述
onBeforeDrag e 在拖动之前触发，返回false将取消拖动。
onStartDrag e 当目标对象开始被拖动时触发。
onDrag e 在拖动过程中触发，当不能再拖动时返回false。
onStopDrag e 当拖动停止时触发。

方法（Methods）

名称 参数 描述
options none 返回属性对象。
proxy none 如果替代元素被设置，返回将被用于拖动的替代元素。
enable none 允许拖动。
disable none 禁止拖动。
