时间调节器 - TimeSpinner

从$.fn.spinner.defaults继承,覆盖默认值$.fn.timespinner.defaults.

timespinner 的创建是基于spinner,很类似于numberspinner 但是显示的是时间值,timespinner允许用户通过点击组件右边的小的spinner 按钮增加值和减少值.

Dependencies

  • spinner

Usage Example

从标记创建timespinner .

使用javascript创建timespinner.


Properties

属性从spinner继承, 以下是timespinner的新增属性.

Name Type Description Default
separator string 这个分隔符在小时和分钟和秒之间. :
showSeconds boolean 定义是否显示秒信息. false
highlight number 初始化的时候高亮字段, 0 = 小时, 1 = 分钟,  0

Events

事件从spinner继承.

Methods

方法从spinner继承, 下面是timespinner重写的方法.

Name Parameter Description
options none 返回options对象.
setValue value 设置timespinner值.

示例代码:

$('#ss').timespinner('setValue', '17:45');  // 设置 timespinner值
var v = $('#ss').timespinner('getValue');  // 获取 timespinner 值
alert(v);
getHours none 得到当前小时值
getMinutes none 得到当前分钟值.
getSeconds none 得到当前秒值.
