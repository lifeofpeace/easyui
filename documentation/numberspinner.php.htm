数字调节器 - NumberSpinner

从 $.fn.spinner.defaults 和$.fn.numberbox.defaults继承,覆盖默认值 $.fn.numberspinner.defaults

numberspinner 创建是基于spinner 和numberbox,可以转换输入的值到不同的类型,例如,numeric, percentage, current, 等等.也可以使用 up/down spinner 按钮滚动到一个期望的值.

Dependencies

  • spinner
  • numberbox

Usage Example

从标签创建numberspinner.

创建numberspinner 和转换数字作为一个货币字符串.

Create numberspinner and format the number as a currency string.


Properties

属性从 spinner and numberbox继承.

Events

事件从 spinner继承.

Methods

方法从spinner继承,下面是numberspinner重写的方法.

Name Parameter Description
options none 返回 options 对象.
setValue value 设置 numberspinner 值.

示例代码:

$('#ss').numberspinner('setValue', 8234725);  //设置值
var v = $('#ss').numberspinner('getValue');  // 得到值
alert(v)
